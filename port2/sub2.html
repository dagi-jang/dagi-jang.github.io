다양한 장르의 공연이 상영되는 메인 공연장입니다. 일반 관객을 위한 대표적인 관람 공간입니다.

좌석배치도

( 장애인석 20석 포함 )

오시는 길

버스 교통 안내

  • 1호선, 7호선 임시역 / 4호선 임의역 중앙차선 정류소에서 133번 (배차간격 약 25분) 탑승 → 마우로예술극장 정류장하차

지하철 교통 안내

  • 버스 : 임의역 2번출구 하차 → 임의역 버스정류장에서 207, 207-1 탑승 → 마우로예술극장 정류장하차 (약 15분 소요)
  • 도보 : 임의역 2번 출구에서 시청방면으로 직진 → 시청앞에서 좌회전하여 직진 (약 15분 소요)

자가용 교통 안내

  • 서울(임의방향) : 임의 → 서부간선도로 → 임의고가교 진입 → 마우로예술극장 방면 진출→ 마우로예술극장 삼거리에서 좌회전 → 100m 직진 후 좌회전
  • 구리(임의방향) : 서울외곽순환 → 임의 IC 출구 → 마우로예술극장 방면 진출 → 마우로예술극장 삼거리에서 좌회전 → 100m 직진 후 좌회전
  • 주차장 이용시 : 일반 방문객 - 30분 이내 무료, 30분 초과 시 10분당 소형기준 300원 부과 / 공연 관람객 - 공연시간(5시간 이내) 선불 1,000원(소형기준)
  • 주차공간이 협소하오니 대중교통 이용을 권장 드립니다.
